Contributors: Su Architect

Tags: download, generate file, xls

== Description ==

This plubin help you:
    - Create an Form which allow users want to download a .xls file (follow their input data)
    - Auto generate xls file which base on user's data
    - Save user's data to the Database for storage or working later.

== Installation ==
This section describes how to install the plugin and get it working.
1. Upload the `dt-download` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. This plugin use an short code "dt-download-xls" to create download-form.
   So please add this short code to the page which you want to show download-form.
